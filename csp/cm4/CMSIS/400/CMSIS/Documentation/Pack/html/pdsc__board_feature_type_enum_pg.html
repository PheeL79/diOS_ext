BoardFeatureTypeEnum

Predefined development board feature categories.

Development board features are categorized using feature types specifying one of the listed types.

Note
  • The name is a descriptive text for a feature. If name is omitted, the Default Name is used.
  • Do not repeat the n or m values in the name. Display on websites follows the rule to show n x name.
