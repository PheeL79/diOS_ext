ExampleProject

A project element is a sequence of environment elements containing the name of the environment and the project file to be loaded.

Parent Chain
project /project
Element Description Type Use
environment Specifies the environment and respective project file name with extension xs:complexType 1..*
Attributes Description Type Use
load specifies the project file with extension (including path relative to folder example attribute if required) xs:string required
name describes the name of the required tool-chain (e.g. uv, iar, etc.) xs:string required
